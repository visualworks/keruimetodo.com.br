== Description ==

Easy to add and display history OR timeline for your WordPress website. This plugin create a "Timeline Slider" menu tab with a custom post type to your wordpress admin side. You can add Title, content and featured image as same like WordPress post.

Timeline and History slider is an advanced WordPress timeline plugin that showcases your life history or your company’s story, WordPress Post in a responsive Horizontal order based on the year and the date of your posts.

View [DEMO](http://wponlinesupport.com/wp-plugin/timeline-history-slider/) | [PRO DEMO and Features](http://wponlinesupport.com/wp-plugin/timeline-history-slider/) for additional information.

= You can use shortcode =

<code>[th-slider]</code>

= Complete shortcode  is =

<code>[th-slider design="design-1" category="8" dots="true" arrows="true" autoplay="true"  autoplay_interval="5000" speed="1000"]</code>
 
= Here is Template code =

<code><?php echo do_shortcode('[th-slider]'); ?></code>

= Use Following parameters with shortcode =

<code>[th-slider]</code>

* **Limit** : [th-slider limit="-1"] (Limit define the number of images to be display at a time. By default set to "-1" ie all images. eg. if you want to display only 5 images then set limit to limit="5")
* **design** : [th-slider design="design-1"] (ie select the design. right now there are 2 designs. design-1 and design-2)
* **Category**: [th-slider category="category_ID"] ( ie Display slider by their category ID ).
* **Pagination and arrows** : [th-slider dots="false" arrows="false"]
* **Autoplay and Autoplay Interval**: [th-slider autoplay="true" autoplay_interval="100"]
* **Slide Speed**: [th-slider speed="3000"]
* **Fade** : [th-slider fade="true" ] (Slider Fade effect. By default effect is slide. If you set fade="true" then effect change from slide to fade ).
* **Loop** : [th-slider loop="true" ] (Enable infinte loop in slider loop="true" OR loop="false" ).

== Installation ==

1. Upload the 'timeline-and-history-slider' folder to the '/wp-content/plugins/' directory.
2. Activate the "timeline-and-history-slider" list plugin through the 'Plugins' menu in WordPress.
3. Add this short code where you want to display slider
<code>[th-slider]</code>
